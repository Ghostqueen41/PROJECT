# Smoking Health Data Analysis (Power BI Project)

## 📊 Project Overview
This project analyzes the impact of smoking on health outcomes using data visualization techniques in Power BI. The goal is to uncover patterns, trends, and insights related to smoking habits and associated health risks.

## 🎯 Objectives
- Understand the relationship between smoking and health conditions
- Identify trends in smoking behavior across different groups
- Highlight key risk factors linked to smoking

## 🛠 Tools Used
- Power BI
- Data Cleaning
- Data Visualization

## 📈 Key Insights
- Smoking patterns vary across different demographics
- Higher smoking rates are associated with increased health risks
- Visual dashboards help in identifying trends quickly

## 📁 Dataset
The dataset includes information such as:
- Smoking status
- Health indicators
- Demographic data

## 📊 Dashboard Features
- Interactive visuals
- Filters for deeper analysis
- Clear representation of trends and comparisons

## 💡 What I Learned
- Building interactive dashboards in Power BI
- Data storytelling using visuals
- Analyzing health-related datasets

# E-Commerce Sales Analysis (Power BI Project)

## 📊 Project Overview
This project focuses on analyzing e-commerce sales data to understand business performance, customer behavior, and revenue trends. The dashboard provides insights that support data-driven decision-making.

## 🎯 Objectives
- Analyze overall sales performance
- Identify top-selling products and categories
- Understand customer purchasing behavior
- Track revenue trends over time

## 🛠 Tools Used
- Power BI
- Data Cleaning
- Data Modeling
- Data Visualization

## 📈 Key Insights
- Certain product categories generate the highest revenue
- Sales trends fluctuate across different time periods
- Customer purchase patterns reveal repeat buying behavior

## 📁 Dataset
The dataset includes:
- Order details
- Product categories
- Sales and revenue
- Customer information

## 📊 Dashboard Features
- Sales overview (total revenue, orders, profit)
- Product performance analysis
- Time-based trends (monthly/weekly sales)
- Interactive filters for deeper insights

## 💡 What I Learned
- Creating business-focused dashboards
- Using Power BI for sales analysis
- Identifying trends and patterns in e-commerce data
